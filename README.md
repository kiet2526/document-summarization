# Document Summarization

## Giới thiệu

Repository này chứa các notebook và script phục vụ việc nghiên cứu và thử nghiệm các phương pháp tóm tắt văn bản (document summarization) và khai thác văn bản. Nội dung tập trung vào việc áp dụng các kỹ thuật NLP khác nhau nhằm phân tích, trích xuất thông tin và tạo bản tóm tắt từ dữ liệu văn bản.

## Cấu trúc thư mục

* TMG.ipynb: Notebook thử nghiệm phương pháp topic modeling phục vụ phân tích nội dung văn bản.
* tmg301m.ipynb: Phiên bản mở rộng hoặc tinh chỉnh của mô hình topic modeling.
* prophetnet.ipynb: Notebook thử nghiệm mô hình ProphetNet cho bài toán tóm tắt văn bản.
* streamlit_text_mining1.py: Script Streamlit dùng để xây dựng giao diện demo cho bài toán text mining và summarization.

## Mục đích sử dụng

* Nghiên cứu và so sánh các phương pháp tóm tắt văn bản.
* Thử nghiệm mô hình NLP trong môi trường notebook.
* Xây dựng demo trực quan cho bài toán text mining.

## Yêu cầu

* Python 3.x
* Jupyter Notebook
* Các thư viện NLP và ML phổ biến (cài đặt tùy theo notebook hoặc script sử dụng).

## Ghi chú

Repository phục vụ mục đích học tập và nghiên cứu. Các notebook có thể được chỉnh sửa hoặc mở rộng thêm tùy theo nhu cầu triển khai thực tế.
